7-Segment Display Decoder

Library: Plexers
Introduced: 2.7.1.1 (Fork)
Appearance:

Behavior

The component has four input pins on its west edge, each input is 1 bit and the lower is the most significant. You have to use these four inputs to enter the binary number and then, in the seven output pins on the east edge, will be set the values for the 7-segment display

All the inputs have to be connected

Pins (assuming component faces east)

West edge (4 inputs, bit width 1)
Input values, indexed from A at the top/west end of the edge to D. D is the most significant bit
East edge (7 outpus, bit width 1)
Output values, indexed from a at the top/east end of the edge to g. Connect these outputs to the corresponding letter of the 7-segment display
North edge (input, bit width 1)
Enable: 0 disables the component.

Attributes

When the component is selected or being added, the arrow keys alter its Facing attribute.

Facing
The direction of the component (its output relative to its input).

Poke Tool Behavior

None.

Text Tool Behavior

None.
