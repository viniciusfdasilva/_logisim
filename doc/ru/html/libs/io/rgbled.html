RGB LED

Library: Input/Output
Introduced: 2.7.1.11 (fork)
Appearance:

Behavior

Each input corresponds to a color (RED, GREEN, BLUE), the RGB LED will display a color composed by the values of each input (multi-bit) or will combine the colors corresponding to a true input (single-bit)

Pins

A RGB LED has three input-pins, each pin corresponds to a color (red, green or blu) and it can be a 1-bit input or a 7-bit input.

Attributes

When the component is selected or being added, the arrow keys alter its Facing attribute.

Facing
The location of the input pin relative to the component.
Multibit?
If true, each pin will be a 7-bit input, this will allow you to use every RGB color combination
Label
The text within the label associated with the component.
Label Location
The location of the label relative to the component.
Label Font
The font with which to render the label.
Label Color
The color with which to draw the label.

Poke Tool Behavior

None.

Text Tool Behavior

Allows the label associated with the component to be edited.
