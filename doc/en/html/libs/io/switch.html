Switch

Library: Input/Output
Introduced: 2.7.1.1 (Fork)
Appearance:

Behavior

Switches its value (0-1) for each click

The Switch component works as the input pin but with just 1-bit values and with a little different appearance

Pins

There is just one pin, a 1-bit output

Attributes

When the component is selected or being added, the arrow keys alter its Facing attribute.

Facing
The location of the input pin relative to the component.
Color
The switch's background color
Label
The text within the label associated with the component.
Label Location
The location of the label relative to the component.
Label Font
The font with which to render the label.
Label Color
The color with which to draw the label.

Poke Tool Behavior

For each mouse click on the component, the output changes its value

Text Tool Behavior

Allows the label associated with the component to be edited.
