Comportamento

Este componente traça a carta de tempo dos sinais de cada entrada conectada, se a habilitação for diferente de 0, atualizada a cada transição de borda do clock. Para se evitar problemas com atrasos (em cada pino de saída), guardará certa quantidade dos últimos valores da oscilação anterior até a última.

Pins

Extremidade oeste, primeiro pino superior marcado por um triângulo (entrada, largura de bits igual a 1)
Clock - quando ativado, enquanto o pino de abilitação não for igual a 0, o diagrama será atualizado.
Extremidade oeste, quantidade variável (entradas, largura de bits igual a 1)
Os valores desses sinais serão apresentados na carta de tempo.
Extremidade sul, pino mais à esquerda (entrada, largura de bits igual a 1)
Enable - quando igual a 0, desabilitará o componenete.
Extremidade sul, segundo pino a partir da esquerda (entrada, largura de bits igual a 1)
Clear - quando igual a 1, a carta de tempo será limpa, e todas as outras entradas serão ignoradas.

Atributos

Número de Entradas
O número de entradas a serem traçadas na carta de tempo
Número de estados
A quantidade máxima de estados traçados na carta de tempo para cada entrada
Bordas do Clock
Escolha para o desenho de linhas verticais tracejadas para as variações de bordas do clock
Cor de borda
A cor para o texto que aparecerá no terminal.
Rótulo
O texto do rótulo associado ao componente.
Localização do rótulo
A localização do rótulo relativa ao componente.
Fonte do rótulo
Tipo de fonte a ser usada no rótulo.
Cor do rótulo
Cor a ser utilizada no rótulo.
