Comportamento

Conduzir um valor simples por um cabo. Para o elemento Fonte, indicado por um triângulo, o valor será um (ou, se o atributo Bits de Dados for maior do que 1, todos os valores iguais à um). Para o elemento Terra, indicado por uma seta formada por três linhas paralelas com tamanhos diferentes, esse valor será zero (ou, se o atributo Bits de Dados for maior do que 1, todos os valores iguais a zero).

A mesma funcionalidade pode ser alcançada pelo uso mais versátil do componente Constante. A única razão para se preferir o uso dos componentes Terra e Fonte é que esses são símbolos eletrônicos padronizados.

Pinos

Há apenas um pino, uma saída cuja largura de bits irá concordar com o atributo Bits de Dados. O componente constantemente irá fornecer à saída o mesmo valor em seu pino: para o componente Terra, a saída será o valor zero, e para o componente Fonte, a saída será o valor um.

Atributos

Quando o componente for selecionado ou estiver sendo incluído, as teclas de Alt-0 até Alt-9 poderão alterar o seu atributo Bits de Dados e as setas irão alterar seu atributo Direção.

Direção
A direção do componente (sua saída em relação à entrada).
Bits de Dados
A largura de bits das entradas e saídas do componente.
